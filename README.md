## Installing the project locally
### Cloning the repo
```batch
$ git clone https://github.com/MoisesLN/moisesfolio.git
$ cd moisesfolio
```

### Installing dependencies and running the project
```batch
$ cd project
$ npm install
$ npm run dev dev
```
The project will then be running at localhost:5173
